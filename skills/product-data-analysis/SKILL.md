name: product-data-analysis

description: Analyze product data to identify trends, growth opportunities, and operational insights.

Use this skill when the user asks about product metrics, user behavior, traffic trends, conversion funnels, or performance analysis.

Examples of when to use this skill:
- Analyze user engagement metrics
- Evaluate traffic sources and conversion rates
- Identify growth opportunities from product data
- Analyze video platform performance metrics
- Investigate user retention and churn
- Compare performance across different content categories

Instructions:

When activated, the agent should:

1. Identify the key metrics involved (DAU, MAU, retention, engagement, conversion rate, traffic sources).
2. Interpret the provided data or scenario.
3. Detect patterns, anomalies, or trends.
4. Provide clear operational insights.
5. Suggest actionable improvements for product growth.

Focus areas:

- User growth analysis
- Retention and churn patterns
- Content performance
- Traffic acquisition
- Conversion funnel optimization
- Engagement metrics

Output format:

1. Key Findings
2. Data Interpretation
3. Potential Causes
4. Strategic Recommendations
